# Обрезка ссылок с помощью VK

Сокращение ссылок с помощью внутренних сервисов VK

# Требования к окружению

Для работы программы потребуются переменная vk_token. Внутри этой переменной хранится сервисный токен приложения, необходимый для запросов к API VK. Получить его можно при создание своего приложения.

# Как установить

Python3 должен быть уже установлен. Затем используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:
```
  pip install -r requirements.txt
  
  pip install requests
  
  pip install python-dotenv
```
Рекомендуется использовать [virtual/venv](https://docs.python.org/3/library/venv.html) для изоляции проекта.

# Пример использования

```
python main.py https://www.kinopoisk.ru/
Сокращенная ссылка:  https://vk.cc/5Ec2DQ

```
